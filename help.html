Endpoints HTTP routes

MethodRouteParams / bodyDescription
GET /windows List open windows with numeric IDs
GET /status Current window and element state
GET /elements [type=ControlType] [id=mapId] [depth=n] [onscreen=true] [match=text] [collapseChains=true] [includePath=true] [properties=extra] [since=hash] Element tree for current window; returns treeHash for polling
POST /find window=<title|id> [id=<automationId|mapId>] [name=<name>] [type=<ControlType>] Select a window and optionally an element - sets the current element pointer
POST /exec action=<action> [value=<input>] Execute an action on the current element
POST /ocr [value=x,y,w,h] OCR the current element or a screen region
POST /capture [action=screen|window|element|elements] [value=id1,id2,...] Screenshot; returns base64 PNG in data.result
GET /uimap Annotated screenshot with colour-coded element overlays
GET /ping Liveness check
GET /health Server health, uptime, and request counters
GET /sysinfo OS and machine info
GET /help [group=<id>] [action=<name>] This reference. Add ?group= or ?action= to filter; append .json for machine-readable output
